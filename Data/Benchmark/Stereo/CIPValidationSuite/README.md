# CIP Validation Suite — external-only registration

`compounds.smi` is intentionally not copied here. At audited revision
`6b9f9db46dadc6749da8234b05164e1e0fb413b9`, the public repository had no
`LICENSE` file and its README supplied citation instructions but no explicit
redistribution grant.

The suite is also a local CIP descriptor-assignment benchmark, not a global
chiral/achiral dataset. If the authors provide a license, it should be added as
a separate task-specific benchmark rather than pooled with the ACS 258 cases.

The central report was generated by reading an exact-SHA external checkout and
runs both whole-molecule settings as non-accuracy diagnostics. Only aggregates,
timings, and case identifiers are frozen; no structure from `compounds.smi` is
redistributed here.
